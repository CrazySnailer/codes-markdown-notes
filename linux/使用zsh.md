###brew安装zsh
```
brew install zsh
```
zsh为默认的shell的话，则输入命令
```
chsh -s /bin/zsh
```
如果想让zsh变得强大，可以编辑~/目录下的.zshrc文件，好消息是这里有一个做好的.zshrc配置，很强大，可以直接拿来用了。
```
cd ~ && wget -O .zshrc http://git.grml.org/f/grml-etc-core/etc/zsh/zshrc && source ~/.zshrc
```
